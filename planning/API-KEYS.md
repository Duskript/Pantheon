|------|---------|
| `~/.hermes/.env` | **Runtime secrets** loaded by Hermes Agent (LLM keys, browserbase, telegram, tavily) |
| `~/pantheon/.env` | Pantheon WebUI env vars (overlay) |
| `~/.hermes/config.yaml` | MCP server configs + hardcoded tokens (GitHub, Composio, auxiliary vision) |
| `~/.hermes/auth.json` | OAuth tokens for OpenAI Codex, Anthropic Claude Code |
| `~/.claude/.credentials.json` | Claude Code credentials (read by Hermes for Anthropic access) |
| `~/pantheon/.env.example` | Template with blank placeholders |
**All `.env` files and `auth.json` are in `.gitignore`. Never commit actual keys.**
