
### Suno Player — Custom Music Player
- **Status:** done
- **Tags:** #pantheon #suno #music #player #webapp
- **Added:** 2026-05-01
- **Notes:** 🎵 Custom SvelteKit web app for playing Suno AI tracks. YT Music-style UI (top nav, grid library, persistent bottom player). Extracts audio + metadata from Suno song pages (no API needed). Local storage via IndexedDB (Dexie.js). Dark theme matching Suno aesthetic, purple accent. Repo: github.com/Duskript/Suno-player. Future: Athenaeum sync for lyrics, styles, full metadata import.
