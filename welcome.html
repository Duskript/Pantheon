API Keys
Pantheon needs at least one LLM provider key. Pick a provider below and paste your key — no terminal needed.
How to get your OpenCode Go key:
  1. Go to opencode.ai/auth
  2. Sign in with GitHub or Google
  3. Click Create API Key → name it "Pantheon"
  4. Copy the key (starts with sk-) and paste above
How to get your OpenRouter key:
  1. Go to openrouter.ai/keys
  2. Sign in with GitHub or Google
  3. Click Create Key → name it "Pantheon"
  4. Copy the key (starts with sk-or-v1-) and paste above

Using Ollama (local models):
  1. Install from ollama.com/download
  2. Run: ollama pull llama3.3
  3. No API key needed — just click Save with any value
Using Ollama Cloud (hosted open models):
  1. Go to ollama.com/settings
  2. Sign in with GitHub or Google
  3. Copy your API key from the settings page
  4. Paste it into .env next to OLLAMA_API_KEY=
