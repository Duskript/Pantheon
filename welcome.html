Pantheon needs at least one LLM provider key. The two best starters are below — pick one.
🟢 OpenCode Go — $10/month, open models, easiest setup
1 Go to opencode.ai/auth
2 Sign in with GitHub or Google
3 Click Create API Key — name it "Pantheon"
4 Copy the key (starts with sk-)
5 Paste it into .env next to OPENCODE_GO_API_KEY=

5 Paste it into .env next to OPENROUTER_API_KEY=
